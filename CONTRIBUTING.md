# Katkıda Bulunma Rehberi (Contributing)

Bu repo herkese açıktır (**Open Source**). Antalya’ya borcunu ödemek isteyen, şehrin geleceğinde söz sahibi olmak isteyen her birey katkıda bulunabilir.

## Nasıl Katkıda Bulunurum?

Kullandığımız terminoloji, klasik yazılım geliştirmeden biraz farklıdır. İşte temel komutlarımız ve anlamları:

### 1. Mirası Anla
Önce şehrin mevcut durumunu, kültürel mirasını ve birikimini anlamalısın.
```bash
git clone antalya-vizyon
```

### 2. Yeni Bir Fikir Geliştir
Şehre değer katacak, sorun çözecek yeni bir proje veya fikir üzerinde çalışmaya başla.
```bash
git branch yeni-fikir
```

### 3. Vefanı Göster
Kişisel tecrübelerini, emeklerini ve çözüm önerilerini kalıcı hale getir.
```bash
git commit -m "vefa"
```

### 4. Gelecekle Paylaş
Bu fikirleri yerel yönetimler, üniversite ve toplumla paylaşarak hayata geçir.
```bash
git push origin gelecege
```

---
*Her "commit" bir tuğla, her "push" bir adımdır.*
